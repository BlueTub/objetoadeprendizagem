PILHA é um tipo de lista linear em que todas as inserções e remoções são feitas num mesmo extremo, denominado TOPO. Cada vez que um novo item é inserido numa pilha é colocado em seu topo. Apenas o item do topo da pilha pode ser removido, devido a essa política de acesso, os itens são sempre removidos na ordem inversa àquela em que foram inseridos. A funcionalidade do topo é a referência ao último elemento adicionado
Pilha é uma das estruturas de dados mais úteis em computação, sendo usada na solução de vários problemas clássicos na área.
Uma pilha é denominada LIFO(Last-In/Fist-Out ou Ultimo-que-Entra/Primeiro-que-sai). Isso significa que o primeiro elemento a entrar, será o ultimo a sair. Imagine uma pilha de pratos onde, você queira pegar o primeiro prato(o que está embaixo da pilha), para fazer isso você precisa retirar todos aqueles que estão acima dele.
Um de seus usos pode ser visto na validação de expressões aritméticas em compiladores.
A pilha é a estrutura de dados de mais fácil compreensão.

FILA é um tipo de lista linear em que todas as inserções são feitas ao final dela e suas remoções são feitas no início, sendo controlado no código internamente por uma variável "tamanho". Em uma Fila a ordem de saída dos itens corresponde diretamente à ordem de entrada.
Fila é uma estrutura de dados muito utilizada na computação, sendo usada em diversas vezes, como por exemplo, em coloração de regiões de imagens (pixel).
Uma Fila é denominada FIFO(First-in/First-out ou Primeiro-que-entra/Primeiro-que-sai). Isso significa que o primeiro elemento a entrar, será o primeiro a sair. Imagine uma fila de banco.
Um de seus usos é alocação de recursos para impressão de documentos em uma impressora (spooler de impressão). Pode ser visto também ao atendimento de processos requisitados a um sistema operacional. FILA
A Fila é uma estrutura de dados de fácil compreensão, um pouco mais complicada que uma Pilha, mas ainda de fácil entendimento.

Lista (também chamado de lista linear) é uma sequência de itens, cujas propriedades estruturais dependem apenas da posição relativa de seus itens.
Lista é a estrutura de dados mais importante, considerando que Pilha e Fila são ramificações suas.
A característica fundamental de uma lista é sua organização unidimensional, que permite dizer precisamente onde ela começa e onde ela termina.
Um de seus usos é no sistema de arquivos de um sistema operacional. O sistema operacional guarda os dados no disco criando uma lista. 
A Lista é uma estrutura de dados de média compreensão.

Árvores são estruturas recursivas, ou seja, removendo a raiz de uma árvore, obtemos uma coleção de árvores. Na árvore binária os dados são armazenados em nós, cada nó é raiz de uma subárvore, onde só é permite no máximo 2 nós. Quando o nó não possue filhos (nós descendentes) é denominado folha, o nó ao topo da árvore é denominado raiz. Para Adicionar um elemento em uma árvore devemos comparar se ele é maior ou menor que sua raiz. Se o número for maior ele deverá ir para direita, se for menor deverá ir para esquerda. Para Remover um elemeto de uma arvore devemos observar seus filhos. Se não existir filhos é só remover o elemento, se existir, o elemento removido será substituido ou pelo menor elemento da subárvore da direita ou pelo maior da subárvore da esquerda.(na ANIMAÇÃO será utilizada a remoção pelo menor da direita)
Árvore é uma estrutura de dados que permite organizar dados de forma hierárquica. 
Árvore é uma estrutura hierárquica composta por nós. A profundidade de um nó é a distância deste nó até a raiz. Um conjunto de nós com a mesma profundidade é denominado nível da árvore. A maior profundidade de um nó, é a altura da árvore.
Um de seus usos é gerador de índices remissivos.
Árvore é uma estrutura de dados de difícil compreensão.
